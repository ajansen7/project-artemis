# Project Artemis — Claude Instructions

## Python Environment

This project uses a virtualenv at `.venv/`. Always activate it before running any Python scripts:

```bash
source .venv/bin/activate
```

Or run scripts directly via the venv interpreter:

```bash
.venv/bin/python script.py
```

Dependencies are in `requirements.txt`. To install:

```bash
pip install -r requirements.txt
```
